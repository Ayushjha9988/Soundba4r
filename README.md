## 💀 About This Project
Ye bot Telegram Voice Chats ke beech ek bridge banata hai jo audio ko **Real-Time** mein process karta hai. Isme **Aggressive Bass**, **Heavy Distortion**, aur **Kernel Presets** ka use kiya gaya hai.

### 🔥 Main Features:
* **[span_0](start_span)Kernel Level Processing:** Aggressive gain aur deep bass boost[span_0](end_span).
* **[span_1](start_span)Presets:** Light, Medium, Heavy, aur Ultimate Kernel presets[span_1](end_span).
* **[span_2](start_span)Audio Tech:** FFT-based EQ, Saturation, aur Bit Crushing ka support[span_2](end_span).
* **Stability:** Bridge manager jo long duration streaming handle karta hai.

## 🛠 Deployment Guide

### 1. Requirements
* API ID & API HASH (From my.telegram.org)
* Bot Token (From @BotFather)
* Pyrogram String Session

### 2. Heroku Setup
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

1. GitHub Repo connect karein.
2. Buildpacks mein `heroku/python` aur `ffmpeg` add karein.
3. Config Vars fill karein aur **Worker** ON kar dein.

## 🎮 Commands
* `/kernel_kernel` - ULTIMATE KERNEL SOUND 🔥
* `/gain <0-500>` - Master Volume Control
* `/bass <0-300>` - Deep Bass Boost
* `/join` - Start the Bridge
